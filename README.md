# Fastify API Demo

## Что делает проект
- `GET /` отдает HTML-страницу с кнопкой `сделать запрос на АПИ`.
- При клике кнопка отправляет `GET /api`.
- Сервер отвечает JSON: `{ "message": "Запрос прошел успешно" }`.
- Клиент выводит это сообщение в консоль браузера.

## Запуск
1. Установите Node.js (вместе с npm): https://nodejs.org/
2. В папке проекта выполните:
   - `npm install`
   - `npm start`
3. Откройте `http://localhost:3000`.
4. Нажмите кнопку и проверьте консоль браузера.
