# Fastify Users Demo

## Что делает проект
- `GET /` рендерит страницу через `Pug` и подключает статику через Fastify.
- `GET /api` возвращает JSON с сообщением `Запрос прошел успешно`.
- `GET /users` показывает список пользователей из массива на сервере.
- `GET /users/create` открывает форму создания пользователя.
- `POST /users` принимает данные формы, добавляет пользователя в массив и перенаправляет на список.

## Запуск
   - `npm install`
   - `npm start`
   - `http://localhost:3000`.
